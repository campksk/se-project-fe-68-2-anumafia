## ⚡ Quick Start

```bash
git clone https://github.com/2110503-CEDT68/se-project-fe-68-2-anumafia.git
cd se-project-fe-68-2-anumafia
npm install
cp example.env.local .env.local   # fill in your values
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

See [Getting Started](docs/GETTING_STARTED.md) for the full setup guide.

| Layer | Technology |
|---|---|
